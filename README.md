# Hotline

## Project Description
**Hotline** is a fast-paced, 2D top-down action shooter built entirely in Python using the Pygame library. Inspired by the classic *Hotline Miami*, the game challenges players to navigate through various close-quarters environments, tactically eliminate enemies, and survive the levels.

## Key Features
* **Multiple Environments:** Battle through 3 distinct, hand-crafted levels (Kitchen, Garage, and Living Room).
* **Interactive Combat:** Fluid top-down aiming and shooting mechanics using the mouse.
* **Custom Game Engine:** Built from scratch using Pygame, featuring a custom map handler, object rendering, and bullet physics.
* **Menu & UI:** Fully functional main menu and victory/defeat result screens.

## Prerequisites
Before running the game, ensure you have the following installed:
* **Python:** version `3.12.1` (or newer)
* **Package Manager:** `pip`
* run using python main.py
